Hearthstone Helper is a web application designed to help users learn the popular online card game Hearthstone.

Gemstone Game

Gemstone Game is a web application where the user attempts to match the random computer score, by clicking the gems. Each gem is given a random, hidden value.

This application was constructed using HTML5, CSS, JavaScript, Bootstrap, and jQuery.
